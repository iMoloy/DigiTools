## 📌 Project Name

**DigiTools**

---

## 📖 Description

DigiTools is a modern, fully responsive digital tools marketplace where users can explore and purchase premium software, templates, and productivity assets in one centralized platform. The application focuses on clean UI, smooth user experience, and efficient workflow enhancement.

---

## 🛠️ Technologies Used

- **React.js**
- **JavaScript (ES6+)**
- **Tailwind CSS**
- **DaisyUI**
- **Toastify (for alerts)**
- **Responsive Design (Mobile-first approach)**

---

## ✨ Key Features

### 1. Dynamic Product Catalog

- Interactive UI for browsing digital tools
- Categorized product listings
- Clean card-based layout

### 2. Functional Shopping Cart System

- Add, remove, and manage products
- Real-time total price calculation
- Smooth checkout flow

### 3. Real-time User Feedback

- Instant notifications using Toastify
- User-friendly alerts for actions
- Improved UX with visual feedback

---

## ⚡ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/iMoloy/DigiTools.git

# Navigate to project folder
cd DigiTools

# Install dependencies
npm install

# Run the development server
npm run dev
```
